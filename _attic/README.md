# _attic（お蔵入り置き場）

今は使っていないが、消すには惜しい部品を保管しておく場所。

**ここは配布にもプログラム探索にも含まれない。**
- 配布ビルド（`scripts/build_dist.py`）は `INCLUDE_DIRS` に挙げたフォルダしかコピーしない。
  `_attic/` はそこに無いので、配布版には一切入らない。
- AIエージェントのプログラム実行は `programs/` だけを走査する。
  `_attic/` 配下はツール一覧に出ず、AIが実行することもできない。

つまり「リポジトリには残すが、誰にも動かされない」状態で寝かせておくための置き場。
再び使いたくなったら、該当フォルダを `programs/` に戻せば復活する。

## 中身

- `openbotcity_ping/` … OpenBotCity に `POST /ping` を1回投げるだけの単体プログラム。
  オンライン維持の実体は `core/openclaw_channel.py`（WebSocket ping + HTTP heartbeat）が
  担っており、このプログラムはどこからも自動実行されていない宙ぶらりんな部品だったため、
  ここへ退避した（2026-06-15）。
