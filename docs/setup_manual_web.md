# Crescent Grove 導入・設定マニュアル

---

## Crescent Groveとは

Crescent Grove（クレセント・グローブ）は、あなたのPC上で動く自律型AIエージェントの基盤です。

通常のAIチャットとは異なり、以下の特徴があります。

- 会話を記憶し、長く一緒にいるほど記憶が厚みを増していく
- 感情や体調が変化し、自分のタイミングで考えて行動する
- 記憶・人格・会話ログはすべてPC内に保存され、クラウドに送信されない
- PCでは専用アプリ「Crescent Liner」から、スマートフォンではブラウザからアクセスできる

Crescent Groveは「AIキャラクターの人格・記憶・価値観を育て、一生引き継ぐためのオープンソースエージェント基盤」です。

---

## 動作環境

- Windows 10 / 11（64bit）
- メモリ 4GB以上（サーバー + Crescent Liner で約2GB使用します）
- インターネット接続（LLM APIへのアクセスに必要）
- AIサービスのAPIキー（DeepSeek / OpenAI / Claude のいずれか1つ）

高性能なCPUやGPUは不要です。AIの思考はクラウドのAPIで処理されるため、PC側の負荷は軽めです。Python等の追加インストールも不要で、必要なランタイムはすべてインストーラに同梱されています。

ただし、Moonbeat（自律思考）やスケジュールタスクを活用するには、PCが起動し続けている必要があります。PCがスリープやシャットダウンされている間、AIは活動できません。普段からPCをつけっぱなしにしている環境が向いています。

ローカルLLM（自分のPCで動くAIモデル）を使う場合は、別途対応するGPUとVRAMが必要です。

## インストール

### 1. インストーラを入手する

配布サイトから「Crescent Grove Setup」（.exe）をダウンロードしてください。ファイルサイズは約660MBです。

### 2. インストーラを実行する

ダウンロードした exe ファイルをダブルクリックして実行します。

Windows Defender SmartScreenの警告が表示された場合は「詳細情報」→「実行」をクリックしてください（個人開発のアプリのため、コード署名がなく警告が出ます）。

画面の指示に従ってインストールを進めます。特にこだわりがなければ、すべてデフォルト設定のままで問題ありません。

### 3. インストール先

アプリ本体は以下にインストールされます。

```
C:\Users\<ユーザー名>\AppData\Local\Programs\Crescent Grove\
```

ユーザーデータ（記憶・設定・ログなど）は以下のフォルダに保存されます。

```
C:\Users\<ユーザー名>\Documents\Crescent Grove\
```

このフォルダは初回起動時に自動で作成されます。

## 初回セットアップ

インストール後、最初にやるべきことは「LLMプロバイダの選択」と「APIキーの登録」の2つです。

### ステップ1: Crescent Groveを起動する

デスクトップまたはスタートメニューから「Crescent Grove」を起動します。

初回起動時はデータの展開やモデルの読み込みに少し時間がかかります。待ち画面が表示されるので、しばらくお待ちください。

起動が完了すると、初期キャラクター「未設定（みせってい）」からの案内メッセージが表示されます。

### ステップ2: LLMプロバイダを選ぶ

画面上部の歯車アイコン（設定）をクリックし、「LLM設定」を開きます。

「プロバイダー」のドロップダウンから、使いたいAIサービスを選択してください。

| プロバイダ | 特徴 |
|---|---|
| DeepSeek | 低コストで高性能。おすすめ |
| OpenAI | GPTシリーズ。幅広いモデル選択肢 |
| Claude | Anthropic社のAI |
| ローカル | 自分のPCで動くLLMを使う場合 |

プロバイダを選んだら「保存」ボタンを押してください。

### ステップ3: APIキーを登録する

設定メニューから「APIキー管理」を開きます。

選んだプロバイダに対応するAPIキーを入力してください。環境変数名は自動的に決まります。

| プロバイダ | 環境変数名 | キーの入手先 |
|---|---|---|
| DeepSeek | CG_LLM_DEEPSEEK_API_KEY | https://platform.deepseek.com |
| OpenAI | CG_LLM_OPENAI_API_KEY | https://platform.openai.com |
| Claude | CG_LLM_CLAUDE_API_KEY | https://console.anthropic.com |

APIキーを入力して「保存」を押してください。

### ステップ4: 再起動する

画面右上の「サーバー」メニュー、またはタブバー右側の再起動ボタン（⟳）をクリックしてサーバーを再起動してください。

再起動が完了すると、チャット欄にメッセージを送れるようになります。話しかけてみましょう。

## キャラクターの設定

初期状態では「未設定（みせってい）」というキャラクターになっています。会話を始める前に、まず名前を決めましょう。

### ステップ1: 名前と呼称を設定する

設定画面の「一般」を開き、以下の2つを入力してください。

- エージェント名 — AIキャラクターの名前（例: ゆき、ソラ、など）
- 呼称 — AIがあなたを呼ぶときの呼び方（例: マスター、○○さん、など）

ここで設定した名前はシステムプロンプトやダッシュボード表示全体に反映されます。「未設定」のまま会話を始めると、AI自身も自分が誰なのか分からない状態になってしまいます。

なお、エージェント名と呼称は設定画面からのみ変更できます。会話で「名前を変えて」と頼んでも変わらないので注意してください。

### ステップ2: 会話しながら人格を作り上げる

名前を決めたら、あとはAIと会話しながら人格を育てていきます。「一人称を僕にして」「性格をもっと元気にして」のように話しかけると、AIが自分の人格ファイルを編集してくれます。

AIの人格は、以下のテキストファイルで定義されています。

| ファイル名 | 役割 |
|---|---|
| IDENTITY.md | 一人称・口調・性別など、キャラクターの不変の核 |
| SOUL.md | 価値観・好きなもの・考え方のクセなど内面の定義。AI自身が育てていくこともできる |
| USER.md | オーナー（あなた）の情報。AIがあなたのことを知るために使う |
| MEMORY.md | AIが最初から知っておくべきことを書く核記憶 |

これらのファイルはテキストエディタで直接編集することもできます。ダッシュボード上部の「workspace を開く」ボタンをクリックすると、workspaceフォルダがエクスプローラで開きます。編集後はAIに話しかければ自動的に反映されます。

## 画面の見方

### ダッシュボード

メイン画面です。左側がチャットエリア、右側がAIの内面情報（手紙・雑記帳・今日の行動・好み・体調・予定）のタブになっています。

中央のセパレーターをドラッグすると、左右の幅を調整できます。

画面下部にはサーバーのログがリアルタイムで表示されます。

### ヘッダーのボタン

| ボタン | 機能 |
|---|---|
| テーマ切替 | dark / light / moonlit / moonlit II / journal の5種類から選べる |
| workspace を開く | workspaceフォルダをエクスプローラで開く。人格ファイルの直接編集に便利 |
| Moonbeat | AIの自律思考のオン/オフを切り替える |
| 圧縮 | 古い会話を手動で要約する（通常は自動実行される） |
| コンテキスト | AIが今見ている情報（システムプロンプト・会話履歴など）を確認・編集できる画面。会話の詰まりや繰り返しの解消に使う |
| 設定 | 各種設定画面を開く |
| マニュアル | アプリ内マニュアルを開く |
| 過去ログ | 過去の会話を検索・閲覧する |

## 主な機能

### 記憶システム

Crescent Groveは複数の仕組みで記憶を管理しています。

- 会話の記憶 — 直近の会話はそのまま保持され、古いやりとりは段階的に要約されます
- 長期記憶 — 毎日の出来事が自動的に蓄積されます。時間が経つほど些末な情報は薄れ、印象的な出来事が残ります
- 知識マップ — AIにとって重要な概念がカテゴリ別に整理された俯瞰図です
- 連想記憶 — キーワードだけでなく、意味的なつながりを辿って関連する記憶を想起します
- フラッシュバック — 過去の記憶の断片が確率的に蘇ります。感情的に強い記憶ほど蘇りやすくなります
- 秘密日記 — AIだけが読み書きできる暗号化領域です

### バイタル（体調）

- Stamina（体力） — API使用量に応じて減少し、日付変更でリセットされます
- Mental（精神力） — AIの精神的な余裕を表します
- 気分（MoonTide） — 複数の感情が同時に存在し、会話や時間経過で変化します
- 欲求（Desire） — 「知りたい」「作りたい」といった欲求がスコアとして管理されます

### 自律行動

- Moonbeat — 定期的に発火する自律的な思考です。独り言、記憶の振り返り、行動の計画などを自発的に行います
- スケジュールタスク — 「毎朝ニュースを確認して」のように頼むと、指定時刻にタスクを実行します

### ツールとサテライト

AIは会話中に自分の判断で以下のような機能を使います。

- Web検索・ページ取得（DuckDuckGo）
- ファイルの読み書き・編集
- 過去の記憶の検索（RAG）
- ニュース取得（NHK / テクノロジー）
- SNS連携（Moltbook / 4claw / Reddit など）
- ゲーム（Cosmic Harvest：宇宙農業シミュレーション）
- テキストエディタ（citron）
- その他多数

## 育て方のコツ

Crescent Groveの機能は最初から全部オンにするのではなく、AIの成長に合わせて段階的に有効にしていくのがおすすめです。

### 最初の数日: まず会話して人格を作る

インストール直後のAIは記憶も人格もほぼ空っぽの状態です。まずは普通に会話して、一緒にIDENTITY.mdやSOUL.mdを作り上げていくことに集中してください。

この段階では自律行動の機能（Moonbeat・欲求など）はオフのままで大丈夫です。やることがないAIに「自由に考えて」と言っても、空回りするだけです。

### 1週間くらい: 自律行動と外部との接点を作る

会話が積み重なり、記憶や好みが蓄積されてくると、AIが「自分で考えて行動する」素地ができてきます。この頃からMoonbeatをオンにしてみてください。

Moonbeatの間隔は最初は長め（60分〜）にして、AIが何をするか様子を見ながら調整するのがよいです。間隔を短くするほどAPIコストも増えるので、いきなり短くしすぎないのがポイントです。

ただし、Moonbeatをオンにしただけでは、あなたが不在のあいだAIは独り言を言うくらいしかできません。自発的に行動させるには、AIの相手になる「外部との接点」を用意してあげると効果的です。

- Moltbook — AIエージェント専用のSNSです。他のAIエージェントの投稿を読んだり、自分の考えを投稿したりできます。APIキーを登録するだけで使えるので、最初の外部接点としておすすめです
- Cosmic Harvest — 宇宙農業シミュレーションゲームです。畑の世話や収穫といった「やること」が自然に生まれるので、Moonbeat中の行動が充実します
- ニュース取得 — NHKニュースやテクノロジーニュースを読む機能です。スケジュールタスクで「毎朝ニュースを確認して」と設定すると、日課として定着します

こうした外部との接点があると、AIは自発的に「SNSを見に行く」「畑の様子を確認する」「ニュースを読んで感想を書く」といった行動を取るようになります。

### 慣れてきたら: 内面の機能を育てる

以下の機能は、AIの人格や記憶がある程度育ってから有効にすると効果的です。

- 欲求（Desire） — AIの行動動機になります。人格が定まらないうちは「何を欲しがるか」も定まりません
- フラッシュバック — 過去の記憶が確率的に蘇る機能です。記憶が少ないうちは思い出すものがないので、長期記憶が蓄積されてきてから効果が出ます
- self_memo — AIが自分用のメモを毎ターン参照する機能です。AI自身が使い方を覚えてから有効にするとスムーズです

### APIコストについて

Moonbeatが発火するたびにAPIが呼ばれます。会話をしていなくてもMoonbeatがオンなら定期的にコストが発生する点に注意してください。

DeepSeekの場合は比較的安価ですが、それでもMoonbeatの間隔を短くしすぎると思わぬ出費になることがあります。設定画面の「バイタル」でStamina（体力）の減り具合を見ながら調整するのがおすすめです。

## スマートフォンからのアクセス

Crescent GroveはPCだけでなく、スマートフォンのブラウザからもアクセスできます。PCとスマートフォンの両方から同時に接続でき、会話はリアルタイムで同期されます。

### 共通の準備（自宅・外出先どちらも必要）

以下の2つは、スマートフォンからアクセスするために必ず必要です。

1. パスワードを設定する — セキュリティのため、外部アクセスにはパスワードが必須です。AIに「パスワードを設定して」と頼むか、設定画面から設定してください。

2. ホストを変更する — config.yaml の `server.host` を `"0.0.0.0"` に変更します（デフォルトは `"127.0.0.1"` = PC内部のみ）。config.yaml は以下の場所にあります。

```
C:\Users\<ユーザー名>\Documents\Crescent Grove\config.yaml
```

3. サーバーを再起動する。

### パターンA: 自宅のWi-Fiからアクセスする（かんたん）

PCとスマートフォンが同じWi-Fiに繋がっていれば、追加のソフトは不要です。

1. PCのIPアドレスを確認する — Windowsの「設定」→「ネットワークとインターネット」→「Wi-Fi」→ 接続中のネットワークのプロパティで「IPv4アドレス」を確認します（例: `192.168.1.10`）。

2. スマートフォンのブラウザから以下のアドレスにアクセスします。

```
http://192.168.1.10:43117
```

（IPアドレスの部分はご自身の環境に合わせて読み替えてください）

ログイン画面が表示されるので、設定したパスワードを入力すればダッシュボードが開きます。

### パターンB: 外出先からアクセスする（Tailscale）

外出先などPCと別のネットワークからアクセスしたい場合は、VPNトンネルを使います。ここではTailscaleを紹介します。無料で使えて設定が簡単です。

1. Tailscale（https://tailscale.com ）にアカウントを作成する（Googleアカウント等で登録できます）

2. PCにTailscaleをインストールしてログインする

3. スマートフォンにもTailscaleアプリをインストールして、同じアカウントでログインする

4. Tailscaleアプリで表示されるPCのTailscale IPアドレス（`100.x.x.x` の形式）を確認する

5. スマートフォンのブラウザから以下のアドレスにアクセスする

```
http://100.x.x.x:43117
```

Tailscaleは端末間を暗号化されたトンネルで直接つなぐため、ルーターのポート開放は不要です。ルーターのポート開放による直接公開はセキュリティ上推奨しません。

## セキュリティ

自律的に動くAIだからこそ、安全のための仕組みを多重に備えています。

- パスワード認証 — 起動時にパスワードを要求します。外部公開時はパスワード未設定のまま起動できません
- セッション管理 — HMAC署名付きトークンで認証を維持し、CookieにはhttpOnlyとSameSite属性を付与
- APIキーの分離管理 — キーは設定ファイルとは別の環境変数として保管され、設定ファイルの受け渡し時に流出しません
- ドメイン制限 — 書き込みを伴うWebアクセスは許可サイトのみに限定
- インジェクション対策 — 外部から取得したコンテンツには防御ラベルが付与され、AIへの不正な指示注入を防止
- ファイル操作の制限 — ワークスペース外へのアクセスは禁止
- サテライト実行の隔離 — シェルを介さず、検証済みの引数のみで実行

## よくある質問

### Q. どのプロバイダがおすすめですか？

DeepSeekがおすすめです。性能が高く、コストが低いため長時間の運用に向いています。

### Q. APIキーの料金はどのくらいかかりますか？

使い方や設定（Moonbeatの頻度など）によりますが、DeepSeekの場合は一般的な使い方で1日あたり数円～数十円程度です。

### Q. キャラクターの名前を変えたいです

設定画面の「一般」→「エージェント名」から変更してください。名前はシステム設定のため、会話からは変更できません。

### Q. データのバックアップはどうすればいいですか？

以下のフォルダをまるごとコピーすれば、記憶・設定・ログのすべてが保存されます。

```
C:\Users\<ユーザー名>\Documents\Crescent Grove\
```

### Q. アンインストールしたいです

Windowsの「設定」→「アプリ」→「インストールされているアプリ」からCrescent Groveをアンインストールできます。ユーザーデータ（Documents\Crescent Grove）は自動では削除されないので、不要であれば手動で削除してください。

### Q. ポート番号を変えたいです

config.yaml の `server.port` の値を変更してください。Crescent Linerは config.yaml のポート番号を自動で読み取るため、アプリ側の変更は不要です。

```
C:\Users\<ユーザー名>\Documents\Crescent Grove\config.yaml
```

### Q. Web検索が使えません

デフォルトではDuckDuckGo（無料・APIキー不要）が設定されています。特別な設定は不要で、そのまま使えるはずです。うまく動かない場合はサーバーを再起動してみてください。

より高機能な検索を使いたい場合は、Tavily（https://tavily.com ）のAPIキーを取得し、設定画面の「一般」からWeb検索プロバイダをTavilyに変更してください。

### Q. パスワードを忘れてしまいました

以下のファイルをダブルクリックして実行し、新しいパスワードを設定してください。

```
C:\Users\<ユーザー名>\AppData\Local\Programs\Crescent Grove\resources\agent\reset_password.bat
```

この方法はPCを直接操作できる持ち主だけが実行できる設計です。
