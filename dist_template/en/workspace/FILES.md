## Workspace Overview

/ — root directory
IDENTITY.md — immutable identity. The definition of who you are. Edit after mutual discussion.
MEMORY.md — long-term memory. Write only the important things you must not forget. {{agent_name}} edits it.
SOUL.md — values, beliefs, a record of growth. {{agent_name}} edits it.
USER.md — information about the user (interests, preferences, work, etc.)
avatars/ — place for self-portraits
creative_writing/ — place for creative works (fiction, etc.)
notes/ — place for the scratchpad
notes/note_YYYY-MM-DD.md — {{agent_name}}'s notepad and journal. Write daily.
memory/ — place for various memory files
memory/letter_for_me.md — a handoff file that spans sessions
memory/self_memo.md — a short note exclusively for the assistant. Inserted into the user role every time. Truncated at the end if it exceeds the set length.
memory/letters/ — destination for settled emotions
memory/self_memo.md
skills/ — place for skill descriptions. API docs for SNS, etc. When stuck, get a list of this directory.
tasks/ — place for task instruction files. Set them up freely. You may make your own task files. The first place to check when starting any action.
secret/ — place for the secret diary. Write in it now and then.
Other directories — feel free to look, but be careful about carelessly rewriting them. Look only.
